To use GPU for cuDNN Installation:
----------------------------------
	- Create Nvidia developer account: https://developer.nvidia.com. Follow the on-screen instruction.
	- Download cuDNN lib: https://developer.nvidia.com/compute/machine-learning/cudnn/secure/v7.1.4/prod/9.0_20180516/cudnn-9.0-linux-x64-v7.1
	- Execute tar xvf cudnn-9.0-linux-x64-v7.1.tgz
	- Copy all contents of cuda directory to human_count/cuda directory
